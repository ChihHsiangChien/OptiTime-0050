# OptiTime-0050: 0050 實時交易模擬挑戰

一個基於 Streamlit 開發的台灣五十 ETF (0050.TW) 實時交易決策與擇時模擬挑戰系統。

## 📌 專案簡介

在股票市場中，我們常聽說「低買高賣」的擇時交易。本專案透過真實的歷史股價數據，帶您進入隨機的交易區間進行實時決策，測試您是否能成功戰勝**「長期持有 (Buy & Hold)」**以及完美的**「上帝視角 (God-View)」**。

---

## ✨ 核心功能與特色

* 📊 **真實歷史數據**：整合 Yahoo Finance API，支援台灣五十 (0050.TW) 自 **2003-06-25 上市至今**的完整歷史調整收盤價。
* 🎮 **實時擇時挑戰模式**：
  * **隨機抽選回合**：系統會隨機選取多個獨立時段（每回合 60 天）。
  * **限制操作起點**：前 10 天預設為歷史參考線，玩家從第 10 天開始執行買賣決策。
  * **流暢前進步長**：支援快速前進 1天、5天 或 20天，觀察後續股價變化。
  * **精緻分批交易**：提供買入（25%、50%、全部）及賣出（25%、50%、全部）之交易深度。
* 📈 **互動式 Plotly 線圖**：動態標記出您的買點、賣點、目前持股成本線與累積資產走勢。
* 🏁 **多維度績效評比**：
  * **玩家報酬**：扣除交易摩擦成本（買入 0.1425%、賣出 0.1425% ＋ 0.3% 證券交易稅，共 0.4425%）後的實際獲利。
  * **長期持有 (B&H)**：以玩家可操作的首日（第 10 天）價格買入並持有至回合結束的報酬率。
  * **上帝視角 (God-View)**：在可操作區間內進行單筆完美交易（精準買在最低、賣在最高）的極限回測績效。
* 📱 **行動裝置最佳化**：針對手機版做介面重構，將操作控制面板固定於底部並適度壓縮間距，且底色依「買入（綠）」、「賣出（紅）」、「時間前進（藍）」進行區隔，實現精緻的單手操控。

## 💻 本機開發與執行指南

本專案需要 Python 3.8+ 環境，請依循以下步驟進行設定：

### 1. 複製專案與啟用虛擬環境
打開終端機（Terminal）並輸入：

* **Linux / macOS**:
  ```bash
  python3 -m venv venv
  source venv/bin/activate
  ```
* **Windows (PowerShell)**:
  ```powershell
  python -m venv venv
  .\venv\Scripts\Activate.ps1
  ```

### 2. 安裝依賴套件
```bash
pip install -r requirements.txt
```

### 3. 啟動 Streamlit 服務
```bash
streamlit run app.py
```
啟動後，瀏覽器會自動開啟 `http://localhost:8501/`。
